Let's now take the first row and try to map how the data is structured:

localhost 127.0.0.1 - - [23/Feb/2014:03:10:31 +0100] 200 - "-" "Apache (internal dummy connection)" VLOG=-

It seems the data is saved in NSCA Common log format: host + ident + authuser + date + request + status + bytes
Parting from the entire row, we'll split it to understand each embedded concept:

       localhost 127.0.0.1 = host address from which the web was accessed

       - - = separator

       [23/Feb/2014:03:10:31 +0100] = date:time and time zone of the register

       "OPTIONS * HTTP/1.0" = user request (method / resource / HTTP protocol)

       200 - = HTTP status code (2xx is a successful response, 3xx a redirection, 4xx a client error, and 5xx a server error) / size of object returned (bytes)

       "-" = separator

       "Apache (internal dummy connection)" - = user_agent browser engine

       VLOG=- - = ?????

So the structure we will use to transpose the data into a pandas dataframe will be structured as this:

       [host] + [IP] + [geolocation] + [date] + [time] + [zone] + [request] + [resource] + [status_code] + [obj_size] + [browser]

- Exercici 3¶

Geolocalitza les IP's.

We can't tell much about the IPs with its original format, but we could use it to extract the geolocation of the user by the time of the login. For this we'll need to import the module ip2geotools and convert each IP to geolocation paramaters.

In [32]:
from ip2geotools.databases.noncommercial import DbIpCity     
web2 = pd.DataFrame(webreg["IP"].sample(100))
web2 = web2[web2["IP"]!= "127.0.0.1"]
web2 = web2.groupby("IP").count().reset_index()

web2["city"] = web2["IP"].apply(lambda x: DbIpCity.get(x, api_key='free').city)
web2["region"] = web2["IP"].apply(lambda x: DbIpCity.get(x, api_key='free').region)
web2["country"] = web2["IP"].apply(lambda x: DbIpCity.get(x, api_key='free').country)
web2["latitude"] = web2["IP"].apply(lambda x: DbIpCity.get(x, api_key='free').latitude)
web2["longitude"] = web2["IP"].apply(lambda x: DbIpCity.get(x, api_key='free').longitude)
